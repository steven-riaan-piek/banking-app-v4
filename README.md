Banking App V4 (C# Console App)

Overview
I enhanced my banking system by adding user authentication with registration and login functionality. This version introduces secure user management and improves the realism of the application.

---

Features
- User registration and login system
- Password-based authentication
- Secure password input (hidden characters)
- Multi-user account management
- Deposit, withdraw, and transfer funds
- Transaction history tracking
- Data persistence using file storage
- Input validation and error handling

---

Technologies Used
- C#
- .NET Console Application
- File handling (System.IO)
- Collections (Dictionary for user management)

---

What I Learned
- Implementing authentication systems (login/register)
- Handling user credentials securely in applications
- Designing more realistic and secure systems
- Managing multiple users with authentication
- Improving overall system structure and user experience
